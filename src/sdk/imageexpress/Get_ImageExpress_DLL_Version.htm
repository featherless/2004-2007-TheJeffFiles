Get_ImageExpress_DLL_Version

The Get_ImageExpress_DLL_Version function returns the version number of the DLL you are working with. If the DLL version returned doesn't equal the version number in your header file, you are using an outdated header/DLL.

Syntax 

float Get_ImageExpress_DLL_Version();
 

Return Value

Returns the version number of Image Express.
 

Remarks

Use the following code to compare your header version to your DLL version: 
if(Get_ImageExpress_DLL_Version()!=IMAGEEXPRESS_VERSION)
    /* exit here*/
