PIXEL_BGR Structure

The PIXEL_BGR contains a single 24 bit pixel.

Syntax 

struct PIXEL_BGR {
                              
   BYTE b,g,r;
   PIXEL_BGR() {}
   PIXEL_BGR(BYTE R,BYTE G,BYTE B) { r=R; g=G; b=B; }
};
 

Members

b
Holds the pixel's blue value.
g
Holds the pixel's green value.
r
Holds the pixel's red value.
