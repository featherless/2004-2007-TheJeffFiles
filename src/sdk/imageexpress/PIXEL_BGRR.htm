PIXEL_BGRR Structure

The PIXEL_BGRR contains a single 24 bit pixel with one byte reserved.

Syntax 

struct PIXEL_BGRR {
                                                           
   BYTE b,g,r;
   BYTE reserved;
   PIXEL_BGRR() {}
   PIXEL_BGRR(BYTE R,BYTE G,BYTE B) { r=R; g=G; b=B; reserved=0; }
};
 

Members

b
Holds the pixel's blue value.
g
Holds the pixel's green value.
r
Holds the pixel's red value.
reserved
Holds a reserved value.
