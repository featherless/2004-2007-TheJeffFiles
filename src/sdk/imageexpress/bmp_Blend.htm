bmp_Blend

The bmp_Blend function blends two bitmaps together using the equation dst*(1-blend)+src*blend.

Syntax 

void bmp_Blend(

   const bmp_DATA* src,
   bmp_DATA* dst,
   float BlendFactor,
   int* ErrorCode
);
 

Parameters

src
[in] The source bitmap to blend with.
dst
[in, out] The destination bitmap to blend with.
BlendFactor
[in] The blending factor, must be 0<=x<=1.
ErrorCode
[out] Pointer to an int value that will hold the error code from this function if there is one. This parameter is optional.
 

Remarks

Supported Bit Depths: 8, 24

Error Codes
