bmp_ChangeBits

The bmp_ChangeBits function converts a bitmap to the requested bit depth. This function requires that you have set a palette if you are converting to 4 or 8 bit color depths.

Syntax 

void bmp_ChangeBits(

   bmp_DATA* data,
   USHORT bits,
   int* ErrorCode
);
 

Parameters

data
[in, out] Pointer to a bmp_DATA structure to be changed.
bits
[in] The new color depth for the image. If the bit depth you requested is already applied to the bitmap, it will not affect the bitmap.
ErrorCode
[out] Pointer to an int value that will hold the error code from this function if there is one. This parameter is optional.
 

Remarks

Supported Bit Depths: 4, 8, 24

If the bit depth you request is the same as the one that the bitmap is already at, the bitmap will not be changed.

If you are converting to 8 or 4 bits, you must set the global palette prior to calling this function using setGlobalPalette.

 

Error Codes

INVALID_POINTER- The pointer passed was invalid and unusable.
UNSUPPORTED_OPERATION- This operation is not supported.
