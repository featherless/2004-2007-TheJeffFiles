bmp_Clear

The bmp_Clear function cleans up the memory used by a bmp_DATA value. Make sure to call this for each bmp_DATA that you use when you are done with it to avoid memory leaks.

Syntax 

void bmp_Clear(

   bmp_DATA* data,
   int* ErrorCode
);
 

Parameters

data
[in, out] Pointer to a bmp_DATA value. This value will be modified to clean up the dynamic memory.
ErrorCode
[out] Pointer to an int value that will hold the error code from this function if there is one. This parameter is optional.
 

Error Codes

INVALID_POINTER- The pointer passed was invalid and unusable.
 

See also

bmp_Open, bmp_DATA
