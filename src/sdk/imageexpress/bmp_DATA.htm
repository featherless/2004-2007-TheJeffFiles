bmp_DATA Structure

The bmp_DATA contains all of the information about a bitmap file.

Syntax 

struct bmp_DATA {
                          
    bmp_HEADER      Header;
    bmp_INFOHEADER  infoHeader;
    PIXEL_BGRR      lookupTable[256];
    UINT            BMPType;
    void*           Data;
};
 

Members

Header
Holds the bitmap header information.
infoHeader
Holds the bitmap information header information.
lookupTable[256]
The 256 color lookup table. Only used for 4 and 8 bit depths.
BMPType
The enumerated bitmap type.
Data
Pointer to the bitmap's color data. Must be typecast to the correct type to be used.
