bmp_Open

The bmp_Open function opens a bitmap (*.bmp) file from your hard drive for editing. The returned bmp_DATA structure can be used for all other bmp_* functions. When you are finished with the bmp_DATA structure, you must call bmp_Clear to clean up the dynamic memory used.

Syntax 

bmp_DATA bmp_Open(

   const char* FileName,
   int* ErrorCode
);
 

Parameters

FileName
[in] Pointer to a null-terminated string saying the name of the file you would like to load.
ErrorCode
[out] Pointer to an int value that will hold the error code from this function if there is one. This parameter is optional.
 

Return Value

If the function succeeds, the return value will be a valid bmp_DATA structure.

If the functions fails, ErrorCode will be set (if you passed a pointer to it) and bmp_DATA will be an invalid value.

 

Remarks

Supported Bit Depths: 4, 8, 24

The current version of image express only supports loading of windows bitmaps.

 

Error Codes

OPEN_FILE_NOT_FOUND- The file couldn't be found.
OPEN_FILE_INVALID_TYPE- This is not a valid bmp file.
OPEN_FILE_UNSUPPORTED_TYPE- This is not a valid windows bitmap or is not of the supported bit depths.
