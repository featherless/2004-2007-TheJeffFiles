bmp_Save

The bmp_Save function saves a bitmap (*.bmp) file to your hard drive.

Syntax 

bmp_DATA bmp_Save(

   const char* FileName,
   const ico_DATA* data,
   int* ErrorCode
);
 

Parameters

FileName
[in] Pointer to a null-terminated string saying the name of the file you would like to save.
data
[in] Pointer to a bmp_DATA structure.
ErrorCode
[out] Pointer to an int value that will hold the error code from this function if there is one. This parameter is optional.
 

Remarks

Supported Bit Depths: 4, 8, 24

The current version of image express only supports saving of windows bitmaps.

 

Error Codes

INVALID_POINTER- The pointer passed was invalid and unusable.
SAVE_FILE_ERROR- There was an error opening the file for writing.
