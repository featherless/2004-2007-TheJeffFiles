bmp_Types Enumeration

The bmp_Types enumeration holds all of the different bitmap types.

Values 

enum bmp_Types {
                        
   BMP_TYPE_UNKNOWN,  // Unknown or invalid bmp type
   BMP_TYPE_BM,       // Windows
   BMP_TYPE_BA,       // OS/2 Bitmap Array
   BMP_TYPE_CI,       // OS/2 Color Icon
   BMP_TYPE_CP,       // OS/2 Color Pointer
   BMP_TYPE_IC,       // OS/2 Icon
   BMP_TYPE_PT,       // OS/2 Pointer
};
