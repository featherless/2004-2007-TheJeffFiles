iPIXEL_BGR Structure

The iPIXEL_BGR contains a single 24 bit pixel.

Syntax 

struct PIXEL_BGR {
                                                          
   int b,g,r;
};
 

Members

b
Holds the pixel's blue value.
g
Holds the pixel's green value.
r
Holds the pixel's red value.
 

Remarks

This structure is used in the blending function for adding up the pixel values and then dividing them.
