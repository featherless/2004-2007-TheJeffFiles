setGlobalPalette

The setGlobalPalette function sets which palette you want to use when converting to a resolution that uses a palette.

Syntax 

void setGlobalPalette(

   paletteType palette
);
 

Parameters

palette
[in] Enumerated value of which palette to set. This must be one of the paletteType enumerations.
 

Remarks

If you do not set a palette before doing conversions from a non-palette bit depth to a bit depth using a palette, it will cause your program to use a blank palette full of zeros.
